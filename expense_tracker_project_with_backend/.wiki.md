Project Summary
The Dynamic Expense Tracker is a web application that helps users manage their personal finances efficiently. It enables users to track income, expenses, loans, and financial goals through an intuitive interface featuring dynamic dashboards and detailed reporting. With support for multiple users, secure authentication, and role-based access, the application prioritizes data privacy. Built with modern technologies like React, TypeScript, and Tailwind CSS, the project aims to improve users’ financial health and decision-making.

Project Module Description
Dashboard: Provides financial summaries, including balance, income, expenses, goals, and loan status with visualizations.
Transaction Management: Users can view, add, edit, and delete transactions, generating unique voucher IDs and PDF vouchers.
Voucher System: Manages unique voucher IDs, allowing users to download and print cheque-sized PDF vouchers.
Financial Reports: Generates reports based on transactions, covering income/expenses, savings rates, and categories.
Loans Module: Tracks loans, monitoring repayment status and balances with dynamic adjustments upon repayment.
User Management: Manages user roles, including viewing, adding, and deleting users.
Settings Tab: Offers options for currency formatting, theme switching, and data backup/restore.
Login Module: Handles user authentication with error handling and registration validation.

Usage
Installation: Ensure you have Node.js installed, then run:
pnpm install
Build: To build the project, execute:
pnpm run build
Run: Start the development server with:
pnpm run dev
